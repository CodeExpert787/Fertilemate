API Routes Explanation

  • / (POST)
    Handles Excel file uploads for model training (requires a gender field).
  • /female_predict (POST)
    Predicts PCOS type for female users and returns a meal plan. Expects JSON user data.
  • /male_predict (POST)
    Predicts health/meal type for male users and returns a meal plan. Expects JSON user data.
  • /image (POST)
    Handles image-related Excel file uploads for image-based predictions.
  • /image_predict (POST)
    Predicts PCOS or fertility assessment based on image and clinical data. Expects JSON data.
  • /send (GET, POST)
    Uploads left and right ovary images, finds the most similar images in the dataset, and returns their features.
  • /fertility_predict (POST)
    Predicts fertile window, ovulation day, and provides cycle insights. Expects cycle info as JSON or form data.
  • /receive_data (POST)
    Receives and processes BBT or cycle data files. Expects a name and a file in the form data.
